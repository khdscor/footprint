# footprint
21년 진행한 캡스톤디자인 결과물을 대부분 수정을 가하여 기능 추가, 다른 프레임워크를 사용하거나 코드를 수정하는 등 리펙토링을 진행한다.
상세 진행 부분은 노션을 참고하길 바란다. 
현재 대부분의 과정을 완료되었고 차후, Redis를 통한 캐싱을 완료하고 배포를 진행할 예정이다.
